# PDFGPT

AI-powered PDF Chatbot built with React, FastAPI, Gemini AI, FAISS, and Sentence Transformers for document-based question answering using Retrieval-Augmented Generation (RAG).

## Features

- Upload multiple PDFs
- Chat with documents
- Gemini AI integration
- FAISS semantic search
- PDF viewer
- Delete uploaded PDFs
- Modern React UI

## Tech Stack

### Frontend
- React
- Tailwind CSS
- Axios

### Backend
- FastAPI
- Gemini AI
- FAISS
- Sentence Transformers
- PyPDF
